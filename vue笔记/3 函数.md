#### 箭头函数

##### 写法

(arguments1,arguments2) => {}

function(arguments1,arguments2){}

arguments1 => {}

function(arguments1){}

arguments1 => 'a'

function(arguments1){
    return 'a'
}

##### 区别

箭头函数的this指向始终指向定义箭头函数的上下文对象

箭头函数不能当作构造函数使用
